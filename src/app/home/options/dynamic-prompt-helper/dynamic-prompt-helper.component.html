Prompt Ideas help you make one prompt that can turn into several different finished prompts. You can preview the possibilities first, then choose whether to keep it flexible or use one exact result.

1. Pick a starting point

Choose a starter, add category chips like Characters or Scenes, or type your own idea.

2. Preview examples

Example outputs appear automatically as you pick a starter or edit the template, and Shuffle Examples refreshes them.

3. Choose how to use it

Use Exact copies one preview into the prompt box. Use Dynamic keeps the choices, so the site picks a version when you generate.

Example _mobian/characters_ in {sunset beach|neon city}, _mobian/styles_

This means: pick a character, pick one of those locations, then add a style.

The category chips add the special-looking parts for you, so you do not need to memorize the format.
